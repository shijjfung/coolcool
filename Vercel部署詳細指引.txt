========================================
  Vercel 部署詳細指引（圖文說明）
========================================

如果找不到 "Redeploy" 按鈕，請按照以下方式操作：

========================================
  方式 1：從部署列表觸發
========================================

1. 前往：https://vercel.com/dashboard
2. 點擊專案：coolcool
3. 點擊標籤：Deployments

4. 在部署列表中：
   - 找到任何一個部署記錄（點擊它）
   - 進入部署詳情頁面
   - 查看右上角，應該會看到：
     * "..." 三個點選單
     * 或 "Redeploy" 按鈕
     * 或 "Deploy" 按鈕

5. 如果看到 "..." 選單：
   - 點擊它
   - 選擇 "Redeploy"
   - 選擇提交或使用快取
   - 確認部署

========================================
  方式 2：檢查專案設定
========================================

1. 在專案頁面，點擊 "Settings" 標籤
2. 點擊左側 "Git"
3. 確認顯示：
   - Connected to GitHub
   - Repository: shijjfung/coolcool
   - Production Branch: main

4. 如果沒有連接：
   - 點擊 "Connect Git Repository"
   - 選擇 GitHub
   - 選擇倉庫：shijjfung/coolcool
   - 完成連接（會自動觸發部署）

========================================
  方式 3：使用 Vercel CLI（終端機）
========================================

在專案目錄執行：

  npx vercel --prod

首次使用需要登入：
  npx vercel login

然後再執行：
  npx vercel --prod

========================================
  方式 4：檢查是否有新提交觸發
========================================

1. 在 Deployments 頁面
2. 查看是否有新的部署正在進行
3. 如果沒有，可能是：
   - Webhook 沒有觸發
   - 需要手動觸發

========================================
