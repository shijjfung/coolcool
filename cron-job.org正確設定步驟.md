# cron-job.org 正確設定步驟（每 1 分鐘）

## 當前問題

您的設定顯示：
- **Crontab 表達式**：`0 0 * * *`（這是每天凌晨 0 點執行，不是每分鐘）

## 正確設定步驟

### 1. 在「執行計劃」區塊

找到 **"自訂"** 選項（您應該已經看到這個選項）

### 2. 選擇「自訂」並輸入 Crontab 表達式

在 **"Crontab 表達式"** 欄位中，將：
```
0 0 * * *
```

改為：
```
*/1 * * * *
```

或直接輸入：
```
* * * * *
```

**這兩個表達式都表示每 1 分鐘執行一次**

### 3. 確認設定

設定完成後，您應該看到：
- **Crontab 表達式**：`*/1 * * * *` 或 `* * * * *`
- **下一次執行**：應該顯示為每 1 分鐘後（例如：`Sunday, November 16, 2025 12:01 AM`）

## 其他常用設定

### 每 3 分鐘執行
```
*/3 * * * *
```

### 每 5 分鐘執行
```
*/5 * * * *
```

### 每 10 分鐘執行
```
*/10 * * * *
```
