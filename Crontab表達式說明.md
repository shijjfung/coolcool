# Crontab 表達式說明

## ❌ 錯誤理解

`3 * * * *` **不是**每 3 分鐘執行一次！

## ✅ 正確含義

`3 * * * *` 表示：
- **每小時的第 3 分鐘執行一次**
- 也就是：00:03, 01:03, 02:03, 03:03... 每小時執行一次
- **一天只執行 24 次**（每小時一次）

## ✅ 正確的表達式

### 每 3 分鐘執行一次
```
*/3 * * * *
```

### 每 1 分鐘執行一次
```
*/1 * * * *
```
或
```
* * * * *
```

## Crontab 表達式格式

```
* * * * *
│ │ │ │ │
│ │ │ │ └─── 星期幾 (0-7, 0 和 7 都表示星期日)
│ │ │ └───── 月份 (1-12)
│ │ └─────── 日期 (1-31)
│ └───────── 小時 (0-23)
└─────────── 分鐘 (0-59)
```

## 常見錯誤對比

| 表達式 | 錯誤理解 | 實際含義 |
|--------|---------|---------|
| `3 * * * *` | 每 3 分鐘 | ❌ 每小時的第 3 分鐘（一天 24 次） |
| `*/3 * * * *` | 每 3 分鐘 | ✅ 每 3 分鐘（一天 480 次） |
| `0 3 * * *` | 每 3 小時 | ❌ 每天凌晨 3 點（一天 1 次） |
| `0 */3 * * *` | 每 3 小時 | ✅ 每 3 小時（一天 8 次） |

## 執行頻率對比

| 表達式 | 執行頻率 | 一天執行次數 |
|--------|---------|------------|
| `3 * * * *` | 每小時的第 3 分鐘 | 24 次 |
| `*/3 * * * *` | 每 3 分鐘 | 480 次 |
| `*/1 * * * *` | 每 1 分鐘 | 1440 次 |
| `0 0 * * *` | 每天凌晨 0 點 | 1 次 |

## 您需要的設定

### 如果每 1 分鐘執行一次：
```
*/1 * * * *
```
或
```
* * * * *
```

### 如果每 3 分鐘執行一次：
```
*/3 * * * *
```

## 重要提醒

- `*/3` = 每 3 個單位（分鐘、小時等）
- `3` = 第 3 個單位（每小時的第 3 分鐘、每天的第 3 小時等）

**記住**：要每 X 分鐘執行，必須使用 `*/X` 格式！
