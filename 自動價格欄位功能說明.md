# 自動價格欄位功能說明

## ✅ 已完成的功能

現在在創建表單時，每個欄位都可以設定價格，客戶端會自動顯示「欄位名稱 價格元」的格式。

## 📊 功能展示

### 創建表單時

當您添加一個欄位（例如「高麗菜口味」）時，系統會自動提供一個價格輸入欄位：

```
┌─────────────────────────────────────┐
│ 欄位標籤：高麗菜口味                │
│ 欄位類型：文字                      │
│                                     │
│ 價格（元，選填）：90                │
│ 💡 設定價格後，客戶端會顯示         │
│    「高麗菜口味 90元」              │
│                                     │
│ ☑ 必填                              │
│ [刪除欄位]                          │
└─────────────────────────────────────┘
```

### 客戶端顯示

客戶在填寫表單時，會看到：

```
┌─────────────────────────────────────┐
│ 欄位          │ 內容                │
├──────────────┼──────────────────────┤
│ 高麗菜口味    │ [輸入框]            │
│ 90元 *       │                      │
├──────────────┼──────────────────────┤
│ 泡菜口味      │ [輸入框]            │
│ 100元 *      │                      │
└─────────────────────────────────────┘
```

## 🎯 使用說明

### 步驟 1：創建表單

1. **前往管理頁面**
   - 點擊「創建新表單」

2. **添加欄位**
   - 點擊「+ 新增欄位」
   - 輸入欄位標籤（例如：「高麗菜口味」）

3. **設定價格**
   - 在「價格（元，選填）」欄位輸入價格（例如：90）
   - 留空表示該欄位沒有價格

4. **完成**
   - 繼續添加其他欄位
   - 每個欄位都可以獨立設定價格

### 步驟 2：客戶端查看

客戶打開表單時，會自動看到：
- **高麗菜口味 90元**
- **泡菜口味 100元**

價格會以藍色粗體顯示在欄位名稱旁邊。

## 💡 功能特點

### 1. 自動顯示

- ✅ 不需要客戶手動輸入價格
- ✅ 價格自動顯示在欄位名稱旁邊
- ✅ 使用藍色粗體，清晰易見

### 2. 靈活設定

- ✅ 每個欄位可以獨立設定價格
- ✅ 可以選擇不設定價格（留空）
- ✅ 支援小數點價格（例如：99.5 元）

### 3. 視覺效果

- ✅ 價格以藍色粗體顯示
- ✅ 與欄位名稱在同一行
- ✅ 不影響表單填寫

## 📝 實際範例

### 範例 1：餐飲表單

## 🔧 技術細節

### 資料結構

```typescript
interface FormField {
  name: string;
  label: string;
  type: 'text' | 'number' | 'select';
  required: boolean;
  options?: string[];
  price?: number; // 新增：價格欄位（可選）
}
```

### 儲存方式

- 價格儲存在表單的 `fields` JSON 中
- 與欄位其他資訊一起儲存
- 不需要額外的資料庫欄位

### 顯示邏輯

```typescript
// 客戶端顯示
{field.label}
{field.price !== undefined && field.price !== null && (
  <span className="text-blue-600 font-semibold ml-1">
    {field.price}元
  </span>
)}
```
